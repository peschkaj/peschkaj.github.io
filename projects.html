Bare Metal OS A bare metal operating system written in the Habit functional programming language.
rustflakes is an integer ID generation service written in Rust. Several integer sizes are supported, but the default integer is a 128-bit integer. Integers are always increasing and unique per computer generating the identifer. By default, a flake contains the 64-bit timestamp (in ms), 48-bits of worker identifier, and a 16-bit discriminator counter that rolls over every millisecond.
Riak Dotnet Client is a client for the Riak distributed database. The client is implemented in idiomatic C# and handles connection pooling, server pool management, and the construction of both search and map-reduce queries against Riak. In addition, only truly exceptional events raise exceptions; all other interaction between the client and user code occurs through Maybe and Either types, minimizing stack unwinding and maximizing performance.
